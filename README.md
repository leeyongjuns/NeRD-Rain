# Rain Removal Experiment based on NeRD-Rain

Overview

This project is based on an existing deep learning model (NeRD-Rain) for rain removal.

The goal is to understand the model, run experiments, and analyze its performance.

Note

This project is based on publicly available code.
I used it for learning and experimentation purposes.
